Smooth

let myRectangle = drawRectangle(8,8,4,4); smooth(myRectangle,0.3);

Allows smoothing a path.

This function will be more readable by passing through a variable to store the path.

let myRectangle = … ;

"let" is a JavaScript keyword used to declare a variable. The term is English and means "let". This code can be translated as: "Let myRectangle be a variable that will store in memory what follows the equals sign."

smooth(myRectangle,0.3);

The first parameter is the path to be smoothed.

The second parameter is the smoothing intensity. The larger the value, the more the path will be smoothed.
